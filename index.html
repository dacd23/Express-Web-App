What is Express?

Express is a JavaScript framework providing many features used to build web applications.

Express Application Project

Set Up Your App's Environment

First, Node.js must be installed in your device. Next, you shall install the Express framework by opening your code's editor terminal and type in npm init. This utility will walk you through creating a package.json file which will contain all your dependencies. Once completed you may install other dependencies using npm install (package name). Upon completition, you may seek the package.json file in your web app folder and modify the "start" section. It controls what will occur once you've initialized your application typing the command npm start.

Routes

Routing determines how an application responds to a client's request to a particular endpoint, which is a path and a specific HTTP request method like GET, POST, PUT or DELETE. The GET method sends a request to obtain data, the POST method sends a request to add data, the PUT method sends a request to replace data and the DELTE method sends a request to delete data. Here you may find more useful information regarding routes and how to use them in your backend application. Reference: Basic Routing Express.
